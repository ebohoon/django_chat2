# faq_chatbot_example
doc2vec, django를 이용한 챗봇 예제입니다.

# 사용 데이터 및 기술
데이터


  * 다음카페 ‘사랑보다 아름다운 실연’에서 자주 나오는 이야기들을 참고한 인공 데이터1) Ÿ 챗봇 트레이닝용 문답 페어 11,876개
  * 일상다반사 0, 이별(부정) 1, 사랑(긍정) 2로 레이블링


사용기술

  * DocVec: WordVex를 문장 단위로 확장한 문서 임베딩 기법
  * konlpy kkma: 한글 형태소 분석 패키지
